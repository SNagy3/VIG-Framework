# Verum Interior Gauge (VIG)

A reproducible research framework for comparative critical-growth laws in regular black-hole interiors.

This repository develops and tests the **Verum Interior Gauge (VIG)** program: a geometric framework for studying the near-critical maximal-slice growth law inside black holes. The current validated core of the project is a comparative analysis of **Hayward** and **Bardeen** regular black-hole models, anchored to the **Schwarzschild** limit and extended through exact reductions, asymptotic analysis, numerical atlas generation, edge scans, and full comparative overlays.

---

## Core result

For the near-critical maximal-slice family, the framework supports the empirical comparative law

\[
V(v)\sim C_{\rm crit}^{(\mathrm{model})}(M,u)\,v,
\]

where:

- \(V(v)\) is the maximal-slice interior volume growth,
- \(C_{\rm crit}^{(\mathrm{model})}(M,u)\) is the model-dependent critical growth constant,
- \(M\) is the mass scale,
- \(u\) is the normalized deformation parameter.

The validated two-model result is:

- **Hayward** and **Bardeen** belong to the same qualitative interior-growth class,
- both reduce to the **Schwarzschild anchor**
  \[
  C_{\rm crit}(M,0)=3\sqrt{3}\pi M^2,
  \]
- both show monotone suppression of the normalized critical-growth law with deformation,
- but **Bardeen suppresses more strongly than Hayward**, with the full-edge comparison showing the normalized Bardeen law falling to about **0.837** of the Hayward law near the top of the overlap.

## Repository purpose

This repository is intended to serve four roles at once:

1. **Paper repository**  
   housing the final manuscript source and compiled PDF.

2. **Code repository**  
   containing the generation, scan, comparison, and aggregation scripts used in the project.

3. **Data repository**

4. **Reproducibility repository**  
   centered on a single master script that aggregates the current validated outputs into a unified results package.

## Scientific structure

The project is organized around the following progression:

### 1. Generic VIG formalism
Given a metric in ingoing Eddington–Finkelstein form

\[
ds^2=-f(r)\,dv^2+2\,dv\,dr+r^2d\Omega^2,
\]

the maximal-slice formalism produces a conserved quantity \(C\), a reduced branch function \(Q(r;C)\), and a critical admissibility condition from which the critical growth law is extracted.

### 2. Schwarzschild anchor
The framework reproduces the known Schwarzschild maximal-slice growth constant:

\[
C_{\rm crit}(M,0)=3\sqrt{3}\pi M^2.
\]

This acts as the anchor for all regular-model deformations.

### 3. Hayward exact law
The Hayward model is treated through exact reduction, asymptotic analysis, atlas generation, and unified summaries.

### 4. Bardeen exact law
The Bardeen model is treated through exact reduction, weak-deformation expansion, near-extremal expansion, atlas generation, dense edge scan, and master summaries.

### 5. Comparative law
Hayward and Bardeen are then compared directly through normalized deformation overlays, shape-function overlays, horizon and critical-slice comparisons, and full-edge ratio analysis.

### 6. Master reproducibility layer
The master driver reads the accumulated validated CSV products and compiles them into a clean reproducibility package.
